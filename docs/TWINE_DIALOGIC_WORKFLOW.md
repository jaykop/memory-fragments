# 📊 Twine + Dialogic 2.0 Workflow Guide

## 🎯 개요

**문제:** Dialogic은 플로우차트 뷰가 없음
**해결:** Twine으로 플로우차트 기획 → Dialogic으로 구현

---

## 🔄 워크플로우

### Phase 1: Twine에서 기획
```
Twine (플로우차트)
  ↓
시각적으로 스토리 구조 설계
  ↓
분기 검증 및 테스트
  ↓
Export (참조용)
```

### Phase 2: Dialogic에서 구현
```
Twine HTML/JSON
  ↓
Dialogic Timeline 작성
  ↓
실제 게임에서 실행
```
